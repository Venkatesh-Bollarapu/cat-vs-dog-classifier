# 🧠 AI Image Classifier Web App (Cat vs Dog)

> A full-stack AI-powered web application that classifies images as **Cat 🐱** or **Dog 🐶** using a deep learning model based on **MobileNetV2**.

---

## 🚀 Features

* 📤 Upload images through a simple web interface
* ⚡ Real-time prediction using Python backend
* 🔗 Seamless integration between Node.js and Python
* 🧠 Deep learning model built with PyTorch
* 🔄 Transfer learning using MobileNetV2

---

## 🛠️ Tech Stack

| Category         | Technology           |
| ---------------- | -------------------- |
| Frontend         | HTML                 |
| Backend          | Node.js (Express)    |
| Machine Learning | PyTorch, MobileNetV2 |
| Image Processing | PIL, torchvision     |

---

## ⚙️ How It Works

1. User uploads an image via the web interface
2. Node.js server receives the image
3. Backend calls Python script (`project1.py`)
4. PyTorch model processes the image
5. Prediction (**Cat 🐱 or Dog 🐶**) is returned instantly

---

## 📦 Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/ai-image-classifier-webapp.git
cd ai-image-classifier-webapp
```

### 2️⃣ Install Dependencies

**Node.js**

```bash
npm install
```

**Python**

```bash
pip install torch torchvision pillow
```

---

## ▶️ Run the Application

```bash
node index.js
```

🌐 Open in browser:

```
http://localhost:3000
```

---

## 🧪 Model Training

The model uses **transfer learning with MobileNetV2** for efficient image classification.

```bash
python trained.py
```
